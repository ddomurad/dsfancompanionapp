Welcome to the Uncharted Dungeons !

Welcome to Dungeon Saga fanmade companion app. This simple application will help you with the deployment of the Uncharted Dungeons while playing with the Invisible Overlord variant.

Quick tabs description:

  • MINIONS SETUP - manage your minions collection.
  • START - star page with basic informations.
  • BOSS - simple tool for generating random bosses.
  • DEPLOYMENT - simple tool for deploying minions while playing full co-op with the Unchared Dungeons variant.
